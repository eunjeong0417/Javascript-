✌ firebase / bootstrap

- firebase로 로그인 기능을 만들었습니다.

- 구글과 깃헙 계정으로 로그인이 가능하며 사용자 정보를 저장해서
  동일 사용자일 경우에는 바로 home 화면으로 이동합니다

- 로그인 화면 디자인은 bootstrap을 사용했습니다
